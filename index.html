서론

건강검진 데이터로 혈압을 예측하는 이유는 여러 가지가 있다. 혈압은 보통 초기에 증상이 없기 때문에 정기적인 혈압 측정을 통해 질병 발생 위험을 초기에 예측하고 대응할 수 있다. 또한, 고혈압은 심장병이나 뇌졸중 등의 심각한 병의 발생 위험을 높일 수 있기 때문에 고혈압 예측은 예방에 중요한 역할을 한다.

이 페이지는 건강검진 데이터를 분석하여 고혈압 여부를 정확하게 예측하기 위한 과정과 결과를 설명한다. 고혈압은 심혈관 질환의 주요 위험 요인 중 하나로, 정확한 예측과 조기 예방이 중요하다. 이를 위해 머신러닝 기법을 사용하여 데이터를 분석하고 모델을 구축한 과정을 소개할 것이다.

본론

① 데이터 전처리
데이터 분석 전, 주어진 데이터를 상세히 살펴보았다. 초기 데이터셋에서는 많은 결측치와 고혈압 여부에 영향을 미치지 않는 특성들이 존재했다. '기준년도', '가입자일련번호', '시도코드', '성별', '연령대코드(5세단위)', '허리둘레', '시력(좌)', '시력(우)', '청력(좌)', '청력(우)', '혈색소', '요단백', '혈청크레아티닌', '흡연상태', '음주여부', '구강검진수검여부', '치아우식증유무', '치석' 등의 feature를 제거하여 데이터를 정리했다.

또한, 각 열의 데이터 분포를 시각적으로 확인하기 위해 박스플롯을 이용했다. 박스플롯을 통해 이상치를 확인할 수 있다.

데이터에는 수축기혈압 200 이상, 이완기혈압 125 이하 또는 30 미만, 식전혈당(공복혈당) 200 이상, 총콜레스테롤 500 이상, 트리글리세라이드 1000 이상, HDL콜레스테롤 200 이상, LDL콜레스테롤 500 이상, 혈청지오티(AST) 500 이상, 혈청지피티(ALT) 500 이상, 감마지티피 1000 이상 등의 이상치가 존재했다. 이에 해당하는 데이터 행들을 제거하고 결측치를 처리하기 위해 categorical 변수는 최빈값, numeric 변수는 중앙값으로 채웠다. 또한, 고혈압 여부를 판단하는 열을 추가했다. 수축기 혈압이 140 이상이고 이완기 혈압이 90 이상인 경우를 고혈압으로 정의하고, 이를 기준으로 고혈압 여부를 나타내는 열을 생성했다. 고혈압일 경우는 1, 고혈압이 아닐 경우는 0으로 지정했다.

또한, 데이터의 기술통계량을 확인하고 결측치가 모두 처리되었음을 확인했다.

다음으로는 데이터의 feature들 간의 관계를 heatmap을 통해 볼 수 있다. 히트맵을 통해 신장(5cm단위)와 체중(5kg단위), 수축기혈압와 이완기혈압, 혈청지오티(AST)와 혈청지피티(ALT)의 관계가 높게 나왔다.

더 깊게 변수 간에 어떤 상관계수를 갖는지 확인하기 위해 산점도를 그려보았다.
세 산점도는 모두 변수간에 양의 상관관계를 가진다.

② 모델 선택 및 머신러닝 과정
본 프로젝트에서는 RandomForest 모델을 사용하여 데이터 학습을 진행했다. RandomForest는 여러 개의 결정 트리를 사용하여 분류 또는 회귀 문제를 해결하는 앙상블 학습 방법이다. 또한, RandomForest는 변수 중요도를 제공하여 각 변수들이 모델에 미치는 영향을 평가할 수 있다.

위의 코드에서는 RandomForestClassifier 모델을 사용하여 데이터를 학습하고 중요 변수들을 도출했다. RandomForest 모델은 결정 트리(Decision Tree)를 기반으로 하는 앙상블 학습 방법 중 하나로, 여러 개의 결정 트리를 사용하여 예측을 수행한다. 각 결정 트리는 데이터의 부분 집합과 특징의 부분 집합을 사용하여 학습된다. 그런 다음, 각 트리의 예측 결과를 집계하여 최종 예측을 도출한다.

위의 그림은 학습된 RandomForest 모델에서 각 변수의 중요도를 나타낸 것이다. 이 중요도는 모델이 예측을 수행하는 데 각 변수가 기여하는 바를 나타낸다. 중요도가 높은 변수일수록 모델이 예측을 수행하는 데 더 큰 영향을 미친다. 이러한 변수 중요도는 모델의 해석 가능성을 높이고, 어떤 변수가 예측에 더 중요한 역할을 하는지 이해하는 데 도움이 된다.

③ 모델 평가
모델의 성능을 평가하기 위해 혼동 행렬(confusion matrix)을 사용했다. 혼동 행렬은 예측된 클래스와 실제 클래스를 비교하여 모델의 성능을 평가하는 데 사용된다. TP(True Positive), FP(False Positive), FN(False Negative), TN(True Negative)의 값을 사용하여 정확도(Accuracy), 정밀도(Precision), 재현율(Recall), F1 점수(F1 Score) 등의 성능 지표를 계산할 수 있다.
위의 코드는 혼동 행렬을 시각화하고 성능 지표를 계산하여 모델의 성능을 평가한다. 이를 통해 모델이 실제 데이터에서 얼마나 잘 예측하는지 확인할 수 있다. 모델의 성능을 평가한 결과는 다음과 같다.
정확도(Accuracy): 전체 샘플 중 맞게 예측한 샘플 수의 비율
정밀도(Precision): 양성으로 예측한 샘플 중 실제 양성인 샘플 수의 비율
재현율(Recall): 실제 양성 샘플 중에서 양성으로 예측한 샘플 수의 비율
F1 점수(F1 Score): 정밀도와 재현율의 조화평균
모델의 성능 지표를 종합적으로 평가하여 모델이 실제 데이터에서 얼마나 잘 작동하는지 판단할 수 있다.

결론

본 프로젝트에서는 건강검진 데이터를 이용하여 고혈압 여부를 예측하는 모델을 구축하고 평가하였다. 데이터 전처리, 모델 학습, 모델 평가의 과정을 거쳐 최종적으로 RandomForest 모델을 사용하여 높은 예측 성능을 달성할 수 있었다. 이 모델을 통해 고혈압 발생 위험이 있는 사람들을 조기에 발견하고 예방 조치를 취할 수 있을 것으로 기대된다. 향후 연구에서는 더 많은 데이터를 수집하고 다양한 머신러닝 기법을 적용하여 예측 성능을 향상시킬 수 있을 것이다.
